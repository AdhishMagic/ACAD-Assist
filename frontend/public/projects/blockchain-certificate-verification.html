Blockchain Certificate Verification

Student Project Submission • Report Preview

Problem Statement

Paper certificates can be forged. This project issues a verifiable credential hash on-chain and provides a verification portal.

Workflow

  • Institute issues certificate metadata
  • Hash stored on-chain via smart contract
  • Verifier scans QR and confirms integrity

Smart Contract Outline

mapping(bytes32 => bool) public issued;
function issue(bytes32 certHash) external onlyIssuer;
function verify(bytes32 certHash) external view returns (bool);
